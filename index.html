Movie & Cinema finder

This project has been developed using the popular React.js framework. Its purpose is to provide users with a convenient way to access information about movies, as well as to locate nearby cinemas based on their current location.

Project Details

  • The Movie & Cinema Finder is a React.js-based single-page application that integrates with various APIs, including OMDB and Geolocation APIs.
  • Users can search for movies by title and view details such as plot, director, and cast, as well as watch trailers and read reviews once top search bar poster has been clicked.
  • The app includes a "Nearby Cinemas" feature that uses the user's location to display nearby cinemas.
